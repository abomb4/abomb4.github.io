# 简介
--------------
二叉树是使用非常广泛的数据结构，在不同领域使用到不同变种的二叉树。

# 概念定义

- 根节点：就是那个根
- 儿子：节点下面跟随的点
- 叶节点：没有儿子的孤苦伶仃节点
- 兄弟：同父节点
- 深度：根节点到最底层节点的层数

```
 +                          +-----------+
 |                          + Root node |
 |                          +-----------+
 |                                |
 |                +---------------+---------------+
 |                |                               |
 |         +-------------+                 +-------------+
 |         | Son of root |                 | Son of root |    （互为兄弟）
 |         +-------------+                 +-------------+
 |                |                               |
 |       +--------+--------+               +------+------+
 |       |                 |               |             |
 |  +----------+        +-----+       +----------+ +----------+
 |  | Leaf 123 |        | 234 |       | Leaf 345 | | Leaf 567 |
 |  +----------+        +-----+       +----------+ +----------+
 |                         |
 |                +--------+
 |                |
 |           +----------+
 |           | Leaf 124 |
 |           +----------+
 v 深度为 4
```
